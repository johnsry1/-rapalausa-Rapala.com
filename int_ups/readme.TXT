*****************************************************************
* A simple form for entering an address and creating a returns 
* shipping label (gif image) at UPS for display in the storefront
*
* Plugs in to Site Genesis
*
*****************************************************************

1) Import Meta Data (ups_meta.xml)

2) Under Site Preferences > Custom Site Preferences > UPS add all necessary configuration data

3) Upload cartridge and add to cartridge path

4) Create directory "UPS" for images in Site library
	- via web dav
	https://your.demandware.instance/on/demandware.servlet/webdav/Sites/Libraries/[YourSiteID]

Access via:
https://your.demandware.instance/on/demandware.store/Sites-SiteGenesis-Site/default/UPS-StartLabel

Enjoy!
